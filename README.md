# Retail Orders Analysis

## Project Overview
End to end data analysis project on retail orders dataset
using MySQL, Excel and Power BI.

## Tools Used
- MySQL — Data storage and querying
- Excel — Data analysis and dashboard
- Power BI — Interactive dashboard

## Dataset
- Source: Kaggle
- Rows: 9994
- Columns: 16

## Key Insights
- Total Revenue: $11.48 Million
- Top Category: Technology ($4.08M)
- Top State: California ($2.3M)
- Best Month: October
- Top Sub Category: Chairs ($1.66M)
- Business grew from $5.68M in 2022 to $5.80M in 2023

## Project Steps
1. Imported data into MySQL
2. Wrote 10 analysis queries
3. Built Excel dashboard with pivot tables
4. Built interactive Power BI dashboard
